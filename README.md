# Anti-India Hate Speech Detection using RoBERTa

## Overview

This project presents a Natural Language Processing (NLP) system for detecting anti-India hate speech on social media platforms. The model analyzes user comments and predicts the probability of hateful content, enabling efficient moderation and supporting safer online communities.

---

## Problem Statement

Social media platforms generate millions of comments every day, making manual moderation difficult. This project aims to automatically identify comments containing anti-India hate speech and assign a confidence score indicating the likelihood of hateful content.

---

## Objective

- Detect anti-India hate speech in social media comments.
- Classify comments using a fine-tuned RoBERTa model.
- Generate a percentage-based confidence score for each prediction.
- Support content moderation by identifying potentially harmful comments.

---

## Features

- Fine-tuned RoBERTa-based text classification
- Confidence score for every prediction
- Binary classification of comments
- Trained on manually labeled social media comments
- High prediction accuracy

---

## Technologies Used

- Python
- RoBERTa
- Hugging Face Transformers
- Machine Learning
- Natural Language Processing (NLP)

---

## Dataset

The dataset was created by collecting comments from multiple social media platforms.

- 1,000+ manually labeled comments
- Custom dataset prepared for model training
- Comments categorized based on hate speech content

---

## Model

- Model: RoBERTa
- Task: Hate Speech Detection
- Training: Fine-tuned on a custom labeled dataset
- Output: Hate speech confidence score (%)

---

## Results

| Metric | Value |
|---------|--------|
| Accuracy | 97–98% |
| Dataset | 1000+ labeled comments |
| Model | RoBERTa |

---

## Project Workflow

1. Collect comments from social media platforms.
2. Clean and preprocess text.
3. Manually label the dataset.
4. Fine-tune the RoBERTa model.
5. Predict hate speech probability.
6. Display confidence score for each comment.
